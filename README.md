An immersive, interactive 3D developer portfolio and high-scoring, Workday-optimized ATS resume console. Built using HTML5, modern vanilla CSS, Three.js (WebGL), FastAPI, and Docker.

## 📈 ATS Optimization Details

The resume (`Fayas_Resume_v7.pdf`) is optimized specifically for **Workday** and similar ATS parsing engines:
* **Layout**: Standard single-column, strictly vertical, table-free design.
* **Separators**: Uses standard bullet points (`•`) instead of piping to avoid parsing splits.
* **Action Verbs**: Diversified start-verbs (*Architected, Created, Designed, Launched, Trained*).
* **Metrics**: 100% metrics-driven descriptions (response latencies, user base size, precision percentages).
